Créez un jeu concours: tirage au sort sur votre site Wordpress en 5 minutes !

== Description ==

Ce plugin permet la mise en place rapide d'un systeme d'inscription à un tirage au sort dont vous etes le createur.
Le plugin determinera lui meme au hasard le ou les gagnants de votre jeu en fonctions des parametres que vous lui aurez fournis.
un controle permet d'empecher toute réinscription.

== Installation ==

This plugin should work out of the box with a standard installation of WordPress.

1 Installer le fichier 'mon_jeu_concours' dans le fichier 'wp-content'/plugin
2 Activez le plugin dans le backoffice de votre site wp
3 Remplissez les champs demandés par le plugin
4 Modifiez si necessaire le 'reglement du jeu' dans le fichier 'mjc_functions.php' dans wp-content/plugin/mon_jeu_concours.
5 Dans la publication de votre choix ( page, articles ...) inserez le shortcode : [mon_jeu_concours]
  le jeux apparaitra à cet emplacement.
6 Au moment de votre choix cliquez sur 'Tirage au sort' dans le menu du plugin
7 contactez le ou les gagnants affichés : atttention si vous recliquez vous procederez à un nouveau tirage au sort.
